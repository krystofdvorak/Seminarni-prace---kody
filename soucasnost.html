Součastnost jazyka C++
V současné době je jazyk C++ stále vyvíjen za standardu C++17. I přestože jazyk C++ není nejmladší ma spousty využití a programátor se znalostí C++ si může přijít na nemalé peníze. Navíc existují spousty C++ kurzů, jako například tato stránka, ale i pokročilejší kurzy. Na youtube najdete pár videí v češtině a tisíce, možná desetitisíce v angličtině. Krom toho existují různé online kurzy s cenami okolo 12$ až 200$. A samozřejmě spousty odborné literatury. Záhodno by bylo poznamenat i široké využití jazyků C a C++ na vysokých školách.
Vhodné by bylo zmínit vývojová prostředí o kterých se budeme bavit v kapitole První program. Důležité je vědět rozdíl mezi IDE a Text Editorem. Jednoduše Text Editor je i například poznámkový blok. Hlavní funkce Text Editoru je aby šel zapsat kód (ideálně i s automatickým doplňováním) a kód poté spustit. IDE je složitější, větší program, který nabíze funkce jako Debugger, různé nastavení GUI (Grafický User Interface) atd. Nepoužívanější je Visual Studio Code, s kterým také budeme pracovat. A teď už se můžeme vrhnout na náš první program.
