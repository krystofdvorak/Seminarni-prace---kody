Zde můžete vidět náš první "Hello world" program. Za chvíli si ho rozebereme.

    #include < iostream >
    using namespace std;

    int main(void)
    {
      cout << "Hello world!" << endl;
      return 0;
    }

#include < iostream > , znamená že překladač (už jsme se o něm bavili) má použít knihovnu iostream . V ní jsou definované základní funkce.

using namespace std znamená používání oboru názvu std . Výklad o oborů názvů ale není součástí této stránky. Pokud jste si ještě nevšimnuli, za každým příkazem krom funkce main a includovaní knihoven se za každým řádkem píšou ;

int main(void) Je základní funkce programu. int je datový typ funkce main a parametr void znamená, že do funkce nepřichází žádne argumenty. Každá funkce je zapsaná do {} závorek.

cout << "Hello world!" << endl; Je základní příkaz cout (console out) je příkaz pro vypsání textu do konzole. Poté následuje << (syntaxe příkazu) , text Hello world a << endl; , konec řádku (end of line)

return 0 je funkce která ukončuje program. Proč zrovna 0? Funkce main je typu int (integer), což je celočíslený typ s rozsahem 4 bajtů. Samozřejmě můžeme napsat i return 0, nebo například return 51456, nejčastěji se používá 1 při úspěšném provedení funkce, 0 při neúspěšném (main je vyjímka)
