Krom jednoduchých proměnných jsou i složité ( user defined ) datové typy. Tyto datové typy si musíte sami nadefinovat, napsat k nim funkce i deklarace, někdy i objekty. Příklad je Array (Pole) , uložení proměnných, struktur, nebo polí (dvourozměrné pole) v paměti hned za sebou. Linked list uložení proměnných v strukturách s jedním nebo dvoumy ukazateli a datovým typem s přístupem pomocí již zmíněných ukazatelů. Zde je odkaz na můj jednoduchý program deklarující Linked List pro práci s ním pomocí funkcí (Tyto funkce jsou objektové.).Poté například Binary Tree , atd. atd. Tyto typy jsou ovšem už pokročilejší částí C++, a kompletní výklad o nich není součástí této stránky. Povíme si ale něco málo o poli.
Graf znázorňuje uložení pole v paměti počítače. Pole ukládá určitý počet proměnných, tentokrát typu int ( ostatní políčka jsou chválně vyplňené jinými proměnnými pro ilustraci )
Zde je kód znázorňující deklaraci pole na obrázku.

    #include < iostream >
    #include "naseknihovna.h"
    using namespace std;

    int main(void)
    {
     int array[3] {58,42,18};

      cout << array[0] << endl;       return 0;
    }

Na konzoli se vypíše číslo 58. Protože pole se indexuje od čísla 0 , číslo 58 má index 0, 42 má index 1, a 18 má index 2.
