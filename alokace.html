Alokace paměti
Nyní si povíme něco o alokaci paměti. Program používá část paměti RAM, která se dělí na Strojový kód , Globální proměnné , Zásobník (Stack) , Volnou pamět a Haldu ( Heap ) . A právě do Haldy se ukládají dynamické proměnné. Dynamické proměnné mají výhodu, že se mohou vytvořit i vymazat. Používají se například při User Defined Datových Typech .

    #include < iostream >
    #include "naseknihovna.h"
    using namespace std;

    int main(void)
    {
     int *i = new int{ 8 };

      cout << *i << endl;
      delete(i);       return 0;
    }

Na konzoli se vypíše číslo 8. Protože k dynamickým proměnným se přistupuje skrz ukazatele ( v našem případě i ), pomocí *i vypíšeme data uložené v proměnné kam ukazuje ukazatel i Ukazatele se deklarují pomocí hvězdičky před názvem proměnné a slouží k ukazovaní na různé datové typy.
delete(i) dynamickou proměnnou uloženou v ukazateli i , smaže.
