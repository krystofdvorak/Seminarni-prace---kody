Implementace Bubble Sort
Bylo by dobré říct si něco o funkcích. Již od začátku znáte funkci main . Existují samozřejmě i jiné funkce např. sqrt() v knihovně math.h Také jsme si zkoušeli nadefinovat svoji vlastní funkci scitej(int a,int b) . Funkce může vracet pole, ukazatel, ukazatel na ukazatel i struktury. Také existují tzv. rekurzivní funkce - funkce které volají sami sebe. Zde je důležité nějakým způsobem funkci ukončit. Příklad rekurzivní funkce pro výpočet faktoriálu:

    long double faktorial(double m)
    {
      if(m==0)
      {
      return 1
      }
      else
      {
      return m * faktorial(m-1)
      }
    }

Faktoriál vypočítáme součinem všech přirozených čísel, která jsou menší nebo rovna číslu m . Je-li m rovno 0, vrátí se hodnota 1. Pokud ne, funkce zavolá sama sebe s parametrem o 1 menším než předtím * číslo m . Jakmile dojde m na nulu, funkce skončí, a začne se vracet zpět. Důležitá věc ohledně rekurzivních funkcí je dávat si pozor aby se funkce dokázala zastavit
