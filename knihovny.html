Propojení knihoven
Nyní si ukážeme, jak si vytvořit vlastní knihovnu s funkcí scitej(int a, int b) pro sčítaní dvou čísel typu int (více o datových tytech v kapitole Data) a propojit ji s našim main.cpp souborem. Budeme potřebovat vytvořit v naší složce dva další soubory, naseknihovna.cpp a naseknihovna.h
Do souboru main.cpp dopíšeme

    #include < iostream >
    #include "naseknihovna.h"
    using namespace std;

    int main(void)
    {
      cout << scitej(5,4) << endl;       return 0;
    }

scitej(4,5) , znamená že funkce scitej() vrátí výsledek 5+4, který se vytiskne, díky příkazu cout
Do souboru naseknihovna.cpp napíšeme:

    #include < iostream >
    #include "naseknihovna.h"
    using namespace std;

    int scitej(int a, int b)     {
      return a+b;
    };

return a+b , znamená že funkce scitej() vrátí výsledek sčítaní dvou čísel, které jsme zadali jako argumenty.
Do souboru naseknihovna.h napíšeme:

    #ifndef   _NASEKNIHOVNA_H_
    #define   _NASEKNIHOVNA_H_

    int scitej(int a, int b);
    #endif

#ifndef , znamená že pokud byla už tato knihovna v programu includována (definována) přeskočí se (vhodné u větších programů)

#define , znamená že pokud tato funkce ještě použita nebyla, definuje se.

#define , znamená že konec této kontroly definice.

int scitej(int a, intb); , je naše funkce z programu naseknihovna.cpp zde ovšem píšeme pouze její deklaraci.
