Implementace Bubble Sort
Nyní si napíšeme a vysvětlíme program pro setřídění pole s 10 prvky od nejmenšího po největší. Použijeme metodu Bubble sort , kterou jsme si již ukazovali zapsanou v Diagramu.

    int* BubbleSort(int* arr, int n)
    {
      bool done = true;
      int temp;
      do{
      for(uint8_t i = 0; i < n; i++)
      {
      if(arr[i] > arr[i+1])
      {
      temp = arr[i];
      arr[i] = arr[i+1];
      arr[i+1] = temp;
      done = false;
      {
      }while(!done)
    return arr;
    }

Funkce Bubble Sort dostane dva argumenty: ukazatel na pole a delku pole. Poté deklarujeme proměnnou typu bool (ano nebo ne), do které přiřadíme hodnotu true. A také pomocnou proměnnou temp. Poté postupně projdeme všechny prvky pole, než narazíme na prvek který je větší než číslo za ním, a prohodíme je. Důležité je, že do proměnné done uložíme hodnotu false. Poté řekneme programu že pokud je v proměnné done hodnota false ( našelo se neseřazené čislo), musí se pole projít znova. Jakmile se pole projde, a nenajde se neseřazené číslo, program vyjde z cyklu do-while, a vrátí ukazatel na pole.
